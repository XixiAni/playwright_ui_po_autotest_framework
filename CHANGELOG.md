# Changelog
所有版本变更记录在此文件中，格式遵循 [Keep a Changelog](https://keepachangelog.com/zh-CN/1.1.0/) 规范，
版本号遵循 [语义化版本](https://semver.org/lang/zh-CN/) 规范。

## [v1.0.0] - 2026-07-09
### Added
- 完成 Playwright + Pytest 核心 UI 自动化测试框架搭建
- 落地五层鲁棒性体系，三层操作容错降级机制，提升动态页面用例稳定性
- 实现多环境 YAML 一键切换，支持定位器增量覆盖
- 落地 YAML 数据驱动参数化，测试数据与业务代码完全解耦
- 集成 Allure 企业级可视化报告，支持失败自动截图、步骤分层展示
- 内置 pytest-xdist 多进程并发执行能力，缩短全量回归耗时
- 实现三级用例标签体系：冒烟/回归/调试，支持命令行灵活筛选
- 支持环境变量占位符脱敏，敏感配置无需硬编码
- 全框架 Locator 实例化改造，对齐 Playwright 官方最佳实践
- 封装公共断言工具集，统一断言规范与日志埋点
- 标准化分层目录结构：配置层/公共工具层/业务PO层/数据层/用例层
- 完善标准化日志体系，支持控制台+文件双输出、按日期拆分

### Changed
- 重构 YamlReader 底层逻辑，统一点式路径取值方法
- 优化 safe_fill/safe_click 容错机制，废弃循环重试改为三层线性降级
- 治理 pytest.ini 编码规范，从根源规避 Windows 平台解码崩溃问题
- 升级 JS 注入为官方参数传参写法，杜绝字符串拼接注入风险

### Fixed
- 修复日志重复打印问题
- 修复 Locator 严格模式多元素匹配报错问题
- 修复并发场景下失败截图同名覆盖问题
- 修复重试配置优先级判断逻辑缺陷
